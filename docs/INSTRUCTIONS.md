# Инструкция по обновлению сайта

1. **Скачайте эту папку (`website_update`)**.
2. **Загрузите файлы в ваш репозиторий**:
   - `index.html`: Замените существующий файл в корне вашего репозитория `website`.
   - `ResearchMate_v2.0.zip`: Загрузите этот файл в корень репозитория.

3. **Проверьте**:
   - Откройте ваш сайт.
   - Нажмите кнопку "Download ZIP" (в верхнем меню или в секции загрузки).
   - Должно начаться скачивание файла `ResearchMate_v2.0.zip`.

Теперь пользователи смогут скачивать собранное расширение прямо с вашего сайта!
